# fom.dll

## Install

1. Copy `dinput8.dll` into the same folder as `fom_client.exe`.
2. Start the game normally.
3. Press F1 in-game to show or hide the `fom.dll` control window.

## Features

- No recoil
- Disable camera shake
- Fire rate control
- Movement speed control
